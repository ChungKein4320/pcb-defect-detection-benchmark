\# Weights



Trained model weights are not committed to GitHub.



After Kaggle training, selected weights can be stored locally here for backup.



Example local files:



\- weights/yolov8n\_best.pt

\- weights/yolov8s\_best.pt

\- weights/faster\_rcnn\_best.pth

\- weights/rtdetr\_best.pt



Git policy:



\- Do not commit `.pt`, `.pth`, `.onnx`, `.engine`, or checkpoint files.

\- Only this README file should be committed.
